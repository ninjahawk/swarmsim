## Model

N agents move on a periodic unit square under four forces each timestep:

| Force | Description |
|-------|-------------|
| **Repulsion** | Short-range ‚Äî keeps agents from overlapping (range 2r‚ÇÄ) |
| **Alignment** | Drives velocity toward mean of neighbors within r_f |
| **Self-propulsion** | Corrects speed toward target v‚ÇÄ |
| **Noise** | Uniform random perturbation in [‚àíŒ∑, Œ∑] |

**Key analytical result:** equilibrium cruise speed is `v_eq = v‚ÇÄ + Œ±/Œº` ‚Äî not `v‚ÇÄ`. Derived from force balance in an aligned flock; verified experimentally to within 0.002 across four Œ± values.

**Default parameters:** N=350, r‚ÇÄ=0.005, Œµ=0.1, r_f=0.1, Œ±=1.0, v‚ÇÄ=1.0, Œº=10.0, Œ∑=0.5, dt=0.01

## Code Architecture

The codebase has two layers.

**Core layer (`model.py`)** ‚Äî object-oriented foundation for all current and future experiments.

```
Flock          ‚Äî holds prey positions, velocities, and all physics parameters
                 flock.evolve(predators=[...]) advances one timestep
Predator       ‚Äî configurable predator agent; strategy='naive'|'encircle',
                 coord_alpha for predator-predator repulsion, enc_radius for encirclement
simulate()     ‚Äî runs a full experiment, returns Phi and distance timeseries
```

New experiments import from `model.py`. To add a new experiment: import `Flock`, `Predator`, and `simulate` from `model.py`, set parameters, call `simulate()`.

**Legacy layer (`flocking.py`)** ‚Äî procedural implementation that predates the OOP refactor. The earlier experiment scripts (`predator.py`, `multi_predator.py`, `encirclement.py`, etc.) import from here. They are retained intact for reproducibility and are not broken by the `model.py` refactor.

The five legacy scripts at root are intentionally retained: every numbered finding F5
through F16 was generated by them, and rerunning them must produce identical figures.
New experiments should not import from them ‚Äî they have no `if __name__ == "__main__"`
guard and will side-effect on import. Use `model.py` instead.

Experiment scripts in subfolders automatically add the project root to `sys.path` so they can import the core library files above.

## Results

### Flock Formation & Coherence

<div align="center">
<img src="figures/phase4_sweeps.png" width="780"/>
</div>

Flock forms reliably above Œ± ‚âà 0.1. With all forces active, the order parameter Œ¶ stays above 0.97 up to noise Œ∑ ‚âà 10, then collapses near Œ∑ ‚âà 20. The alignment force makes the system dramatically more noise-resistant than the repulsion-only case.

### Phase Transition Analysis

<div align="center">
<img src="figures/phase_transition_scaling.png" width="780"/>
</div>

Finite-size scaling across N = 25‚Äì200 (with compactness fixed via r‚ÇÄ = ‚àö(C/œÄN)) shows KE/N curves are N-independent and susceptibility œá = N¬∑Var(KE/N) increases monotonically at every tested compactness value (C = 0.10‚Äì0.78). No diverging peak appears anywhere. The solid-to-fluid transition is a **smooth crossover** throughout ‚Äî a consequence of the soft repulsion potential used in this model. A true phase transition would require hard-core exclusion.

### Predator Strategy Hierarchy

<div align="center">
<img src="figures/predator_2_coherence.png" width="560"/>
</div>

Flocking prey maintain Œ¶ ‚âà 1.0 under sustained predator pressure; non-flocking prey scatter to Œ¶ ‚âà 0.1 almost immediately. A minimum evasion buffer distance (~0.10) persists regardless of predator aggression.

With multiple predators, flock coherence remains high (Œ¶ > 0.97) because all predators independently target the same center of mass and co-localize (measured separation ~0.001). This self-undermining behavior was confirmed in `evasion_analysis.py`.

Adding predator-predator repulsion (coordinated predators) successfully spreads predators out ‚Äî separation rises to 0.29 ‚Äî but Œ¶ never drops below 0.92. The flock absorbs distributed pressure from multiple directions without breaking.

<div align="center">
<img src="figures/encircle_3_breaking_threshold.png" width="640"/>
</div>

**Encirclement** (each predator assigned a fixed compass angle, targeting CoM offset by R_enc in that direction) is the first strategy to substantially disrupt the flock. At n_pred = 6, Œ¶ drops to 0.77. The minimum predator-prey distance falls to 0.050 vs 0.105 for naive predators.

<div align="center">
<img src="figures/frag_2_cluster_stats.png" width="640"/>
</div>

This Œ¶ = 0.77 does not represent dissolution. `fragmentation.py` shows that encirclement **divides** the flock: predators compress agents spatially (60 clusters ‚Üí 24 larger clusters) while splitting them directionally ‚Äî each sub-flock escapes through a gap between predators and remains internally coherent (sub-flock Œ¶ = 0.997). This is analogous to wolf-pack herding.
